# Super Mario: Gravity Chaos 🍄🌌

A precision platformer built with the **Antigravity Engine**. Experience the classic Mushroom Kingdom with a glitchy, gravity-defying twist.

## 🚀 How to Play
1. Open `index.html` in your browser.
2. Use **A / D** or **Arrow Keys** to move.
3. Use **W / SPACE** to jump.
4. Collect **Coins** to increase your score.
5. Watch out for **Gravity Stars**—they will flip your world upside down for 5 seconds!
6. Beware of **Hidden Blocks** and **Falling Platforms**... this realm is unstable.

## 🛠️ Tech Stack
- **Engine:** Custom Vanilla JS Canvas Engine (Antigravity)
- **Styling:** Vanilla CSS (Glassmorphism + Pixel Art)
- **Assets:** AI-generated 32x32 sprites
- **Resolution:** 800x600 (4:3 Retro)

## 📁 Project Structure
```text
/mario-gravity-chaos
├── /assets          # AI-generated Sprites & Backgrounds

├── index.html       # Entry Point
└── style.css        # Premium UI Design
```

This project was designed to showcase high-performance canvas rendering and gravity-based physics state management. The "Troll" elements are integrated directly into the collision matrix.
